# ocicl 2.15.1 Release Notes

## Bug Fixes

- Fixed `ocicl update --check` to only check for updates without downloading
- Added Linux standalone tarball to release assets for self-update support

## Installation

Download the appropriate package for your system from the [releases page](https://github.com/ocicl/ocicl/releases/tag/v2.15.1):

**Linux:**
- **Standalone**: `ocicl-2.15.1-linux-amd64.tar.gz`
- **Fedora/RHEL/CentOS**: `ocicl-2.15.1-1.*.x86_64.rpm`
- **Debian/Ubuntu**: `ocicl_2.15.1-1_amd64.deb`

**Windows:**
- **Installer**: `ocicl-2.15.1-setup.exe` (recommended)
- **MSI**: `ocicl-2.15.1.msi`
- **Chocolatey**: `choco install ocicl`
- **ZIP**: `ocicl-2.15.1-windows-amd64.zip`

**macOS:**
- **ARM64 (Apple Silicon)**: `ocicl-2.15.1-macos-arm64.tar.gz`
- **x64 (Intel)**: `ocicl-2.15.1-macos-x64.tar.gz`
- **Homebrew**: `brew install ocicl`
