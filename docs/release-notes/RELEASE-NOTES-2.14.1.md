# ocicl 2.14.1 Release Notes

## Bug Fixes

- Fixed linter incorrectly transforming `zerop` to `(= x 0)` in vendored pure-tls test file

## Installation

Download the appropriate package for your system from the [releases page](https://github.com/ocicl/ocicl/releases/tag/v2.14.1):

**Linux:**
- **Fedora/RHEL/CentOS**: `ocicl-2.14.1-1.*.x86_64.rpm`
- **Debian/Ubuntu**: `ocicl_2.14.1-1_amd64.deb`

**Windows:**
- **Installer**: `ocicl-2.14.1-setup.exe` (recommended)
- **MSI**: `ocicl-2.14.1.msi`
- **Chocolatey**: `choco install ocicl`
- **ZIP**: `ocicl-2.14.1-windows-amd64.zip`

**macOS:**
- **ARM64 (Apple Silicon)**: `ocicl-2.14.1-macos-arm64.tar.gz`
- **x64 (Intel)**: `ocicl-2.14.1-macos-x64.tar.gz`
- **Homebrew**: `brew install ocicl`
